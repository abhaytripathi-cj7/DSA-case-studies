# DSA-2 Case Studies Project

## 📌 Project Overview

This repository contains **real-world Data Structures and Algorithms (DSA-2) case studies with Java implementations** developed as part of the academic project for **K L Deemed to be University**.

The project focuses on solving practical industry-based scenarios using advanced data structures and algorithms such as:

* **Binary Search Trees (BST)**
* **AVL Trees**
* **B+ Trees**
* **Minimum Spanning Trees (MST - Kruskal’s Algorithm)**

Each case study includes:

* Real-world problem statement
* Algorithmic analysis
* Step-by-step solution
* Java implementation
* Complexity evaluation

## 🧩 Case Studies Included

### 1️⃣ Binary Search Tree (BST) & AVL Tree Indexing

**Scenario:** Healthcare telemetry routing system.

### Features:

* BST construction
* AVL tree balancing
* Rotations (LL, RR, LR, RL)
* Tree deletion operations
* SLA lookup latency analysis

### Concepts Used:

* Binary Search Trees
* AVL Trees
* Tree Rotations
* Time Complexity Analysis

---

### 2️⃣ B+ Tree Range Query Optimization

**Scenario:** Food delivery platform database indexing system.

### Features:

* B+ Tree indexing
* Range query optimization
* Leaf node traversal
* Query performance analysis
* Production-ready range count method

### Concepts Used:

* B+ Trees
* Database Indexing
* Range Queries
* Performance Optimization

---

### 3️⃣ Minimum Spanning Tree (MST) using Kruskal’s Algorithm

**Scenario:** Power grid network optimization.

### Features:

* Kruskal’s MST algorithm
* Disjoint Set (Union-Find)
* Cost minimization
* High-availability network design

### Concepts Used:

* Graph Theory
* Kruskal’s Algorithm
* Minimum Spanning Tree
* Disjoint Set Forest
